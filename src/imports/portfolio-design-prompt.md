Fade In + Slide Up: Duration 0.6s, delay 0.2s, ease-out

Elements: Text sections, skill cards, project cards
Slide distance: 30px from bottom


Fade In + Scale: Duration 0.5s, delay 0.1s, ease-out

Elements: Icons, badges, stat boxes
Scale range: 0.8 to 1


Slide In from Left: Duration 0.6s, ease-out

Elements: Timeline left items, left-side content
Slide distance: 50px from left


Slide In from Right: Duration 0.6s, ease-out

Elements: Timeline right items, right-side content
Slide distance: 50px from right



Hover Animations (On User Interaction)

Lift Effect: transform: translateY(-8px), duration 0.3s

Applied to: Cards, buttons, links


Glow Effect: Box shadow with color #7C3AED or #06B6D4, duration 0.3s

Applied to: Buttons, important elements


Scale Effect: scale(1.05) on button hover, duration 0.2s

Applied to: CTA buttons


Color Transition: Smooth color change, duration 0.3s

Applied to: Links, navigation items



Continuous Animations (Loop)

Floating Animation: Up-down movement, 3s duration, infinite

Translate Y: -5px to 5px
Applied to: Hero section shapes, profile image


Gradient Shift: Color transition, 5-7s duration, infinite

Applied to: Hero background, borders, accents


Pulse Effect: Opacity/scale pulse, 2s duration, infinite

Applied to: Timeline dots, icons, badges


Spin Animation: 360° rotation, 8-10s duration, infinite

Applied to: Background shapes, decorative elements




🎥 MICRO-INTERACTIONS
Button Interactions
States:
- Default: Background `#7C3AED`, text white
- Hover: Scale 1.05, glow effect with cyan shadow
- Active/Click: Slight scale down 0.98
- Loading: Animated loader inside button (spinner animation)
Link Interactions
States:
- Default: Color `#06B6D4`
- Hover: Color changes to `#7C3AED`, underline grows from left
- Active: Color remains, underline stays visible
Form Elements (If contact form present)
- Input focus: Border color changes to `#7C3AED`, glow effect
- Input valid: Border color changes to `#10B981` (success)
- Input error: Border color changes to `#EF4444` (error), shake animation
Scroll Effects
- Parallax on hero background (subtle, 0.5x scroll speed)
- Progress bar at top indicating page scroll percentage
- Smooth scroll behavior globally

📱 RESPONSIVE DESIGN BREAKPOINTS
Desktop (1440px and above)

Full animations active
3-column grids where applicable
Full hover effects

Tablet (768px - 1439px)

Reduced animation complexity (keep smooth, reduce float distance)
2-column grids
Touch-friendly button sizes (44px minimum)

Mobile (< 768px)

Single column layouts
Simplified animations (remove parallax, reduce float distance)
Tap-to-interact instead of hover
Larger text sizes for readability
Button sizes: 48px minimum for touch targets

🎯 DESIGN PRINCIPLES TO FOLLOW

Consistency: Use the same color palette, typography, and spacing throughout
Contrast: Ensure text is readable against backgrounds (WCAG AA minimum)
White Space: Don't overcrowd sections; use breathing room
Accessibility: Color + icons for information (not color alone)
Performance: Optimize animations for smooth 60fps on all devices
Modularity: Design reusable components (cards, buttons, etc.)


🚀 ANIMATION PERFORMANCE TIPS

Use transform and opacity for animations (hardware-accelerated)
Avoid animating width, height, left, top (costly)
Use CSS animations over JavaScript where possible
Throttle scroll events for scroll-based animations
Consider prefers-reduced-motion for accessibility
